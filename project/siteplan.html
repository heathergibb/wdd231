Site Name:Rocky Mountain Camping | Alberta

I chose this name because the main purpose of the page is to view some info about camping in the mountains of Alberta.

Site Purpose:The site provides info about a few campgrounds that I like in Alberta's Rocky Mountains. It will also include some basic info about the national/provincial parks that the campgrounds are located in (Banff, Jasper and Kananaskis).

Scenarios:

  • What are some things to do in the area?
  • What is the current weather like at each location?
  • What amenities are included at each campsite?
  • Request to subscribe to a newsletter.
